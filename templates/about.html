About CitizenAI

CitizenAI is an AI-powered chatbot designed to help citizens get quick responses to government-related queries. The platform also analyzes sentiment from user feedback to better understand public opinion.

Built with Flask, MongoDB, Hugging Face LLMs, and Tailwind CSS, CitizenAI enables transparent communication between governments and citizens using cutting-edge natural language technology.

Key Features

  • AI-Powered Responses: Get instant answers to government service questions using advanced language models
  • Sentiment Analysis: Your feedback is analyzed to help improve public services
  • Dashboard Analytics: View insights and trends from citizen interactions
  • Secure Platform: Built with modern security practices and data protection
  • Government Services Coverage: Information about licenses, voting, taxes, benefits, and more

Technology Stack

  • Backend: Flask web framework with SQLAlchemy
  • AI Integration: Hugging Face transformers and language models
  • Analytics: TextBlob for sentiment analysis
  • Database: PostgreSQL for production, SQLite for development
  • Frontend: Modern CSS with beautiful gradient backgrounds

Getting Started

Ready to get better government services? Create your account and start asking questions right away. Our AI assistant is trained to help with:

  • Driver's licenses and vehicle registration
  • Voting registration and election information
  • Tax filing and payment assistance
  • Social Security and Medicare benefits
  • Business licenses and permits
  • Legal aid and court procedures
